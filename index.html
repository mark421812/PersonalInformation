關於我

我的故事

我從 2021 年開始自學 Web 與後端開發，短短幾年內從助理工程師一路成長為資深系統分析師，主要負責 企業內部系統的架構設計、需求分析與技術整合。在專案中，我常在商業部門與技術團隊之間協調， 將抽象需求轉換為具體的系統設計與實作規格。

技術上熟悉 C# / .NET、React + TypeScript、Oracle 與 PostgreSQL，並實際參與過多資料庫同步、 權限與角色管理平台、企業入口網站重構等大型專案。我特別重視程式碼可維護性， 喜歡使用分層架構與 Clean Architecture 來管理系統複雜度。

工作之餘，我會研究投資理財與量化模型、跑步訓練、露營，以及持續學習軟體架構、 雲端與 DevOps 相關技術，讓自己在職涯上維持成長與彈性。
